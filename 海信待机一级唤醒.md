You are working on the file `PJ-hisense/hisense_litewakeup_analyze_wakeup.py`.

Task:
1. Add a command-line interface that accepts `--input-dir`, `--wav-output-dir`, `--log-output-dir`, `--lite-wakeup-cmd`, `--wakeup-params`, and `--sample-rate`.
2. Batch process raw `.pcm` files from the input directory as 16-bit little-endian stereo.
3. Extract the first channel and save it as mono WAV files at the specified sample rate.
4. Run `lite-wakeup` on each generated WAV using the provided wakeup parameters.
5. Save each tool invocation output to a `.log` file named after the WAV base name.
6. Analyze each saved log to count valid wakeup events where `hai xin xiao jv` is immediately followed by `kai ji` and the `frame` difference is less than 100.
7. Print per-file valid wakeup counts and a total count.

Use standard Python libraries unless the workspace already includes a preferred audio library. Keep the script compatible with Python 3 and validate syntax with `python3 -m py_compile`.
